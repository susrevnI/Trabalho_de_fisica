# Trabalho de Física

## Criação de uma Web App para cálculo de sistemas mecânicos com pólias e blocos.

 * **Protótipo** => Uso de HTML e CSS para apresentação gráfica, e JavaScript para captura e cálculo de dados.

 * **Versão 1.0** => Uso de React+JavaScript para apresentação gráfica, captura e cálculo de dados.

 * **Versão 1.1** => Apresentação gráfica por meio de diagrama estático do sistema.

 * **Versão 1.2** => Apresentação gráfica por meio de diagrama do sistema com física aplicada

---

### Instalar dependencias
`$ npm install`

### Rodar projeto
`$ npm run dev`
